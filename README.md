# EDA Workspace

This repository is a working space for exploratory data analysis (EDA) across multiple projects.

It is intended to hold:

- early project exploration notebooks
- throwaway/prototype code used to test ideas quickly
- starter analysis scaffolding before code moves into more formal project repos

## Organization

The repo is organized by project folder.

Current project folders include:

- `DoE jobs`
- `CII2`

Each project folder can contain its own notebooks, helper scripts, and notes.

## Typical Workflow

1. Create or open a project folder.
2. Add exploratory notebooks/scripts.
3. Iterate quickly on ideas and data checks.
4. Promote reusable code to production-oriented repositories when ready.

## Environment

Recommended Python version: 3.10+

Common packages:

```bash
pip install pandas matplotlib openpyxl jupyter
```
